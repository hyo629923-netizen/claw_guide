1

建立 Telegram Bot 並取得 Token

前往 @BotFather 建立你的機器人，並獲取 API Token。

  • 傳送 /newbot 給 BotFather。
  • 設定機器人的顯示名稱與 username。
  • 完成後 BotFather 會給出一串 API Token（例如：123456:ABC-DEF...）。

BotFather 建立 Token

複製並保存 Token 到表單中。

2

讓 bot 收到訊息並取得 Chat ID

開啟你剛建立的 bot 對話，送出第一則訊息後，回到表單點擊「自動取得 Chat ID」。

GitHubClaw 是透過 Telegram 的訊息更新紀錄取得 Chat ID。

開啟 bot 資訊頁

點擊 bot 的 username。

開啟 bot 對話

在 bot 資訊頁中點擊「Message」。

傳送第一則訊息

第一次進入聊天室時，點擊「START」。

確認 bot 已收到訊息

只要聊天室中已經有你送出的訊息即可。

自動取得 Chat ID

回到 GitHubClaw 點擊「自動取得 Chat ID」。
