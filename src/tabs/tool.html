補充：建立小龍蝦工具 (The Lobster Tools)

這是小龍蝦的「技能包」。除了基礎的 GitHub 操作，我們還可以串接外部強大的工具（如 Felo 搜尋、Gemini 視覺模型），讓小龍蝦更有戰力。

外部工具 API：

設定 Felo 或 Gemini 的 API Key，讓小龍蝦具備上網搜尋或處理複雜圖像的能力。
